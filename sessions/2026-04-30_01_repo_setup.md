# 2026-04-30 repo setup

## Context

Created a reusable research repository structure for a CG + AI paper project on wind-driven deformable 3D Gaussian Splatting.

## Decisions

- Use `ideas/` for living research sketches and decision logs.
- Use `code/` for implementation and runnable prototypes.
- Use `paper/` for the eventual manuscript.
- Use `experiments/` for experiment-specific notes.
- Use `assets/` for diagrams, videos, and visual materials.
- Use `sessions/` to track AI-agent work sessions because extension-level chat history may mix multiple repositories.
- Use the conversation prefix `[Wind3DGS | area | optional_exp]` for this project.
- Keep the reusable research workflow in the parent workspace as `../RESEARCH_PROJECT_GUIDE.md` so sibling research repos can share one guide.
- Keep reusable document templates in `../templates/research_project/` and localize them per repository.
- Track template versions with `template-version` metadata and compare local files against `../templates/research_project/TEMPLATE_MANIFEST.md` at the start of meaningful work.
