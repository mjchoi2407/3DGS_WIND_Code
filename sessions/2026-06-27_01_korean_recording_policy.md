# 2026-06-27 한국어 기록 원칙 반영

## 배경

사용자가 앞으로 프로젝트의 모든 기록을 한국어로 남기도록 요청했다. 대상에는 session 기록, README, 작업 로그, 스크립트가 남기는 설명성 로그가 포함된다.

## 변경 내용

- `../AGENTS.md`에 프로젝트 전체 기록 언어 규칙을 추가했다.
- `AGENTS.md`에 코드-side 기록 언어 규칙을 추가했다.
- 새로 작성하거나 갱신하는 코드-side 기록은 한국어를 기본 언어로 사용하도록 명시했다.

## 예외

- 명령어, 파일 경로, 코드 식별자, API 이름, 논문/라이브러리의 공식 영문 명칭은 원문을 유지한다.
- 외부 도구가 출력한 에러 메시지, 테스트 로그, 라이브러리 로그는 원문을 유지할 수 있다.
- 사람이 덧붙이는 요약과 해석은 한국어로 작성한다.

## 비고

기존 영문 기록은 별도 요청이 없는 한 소급 번역하지 않는다.
